※ 개선 사항

1. 난이도 선택
→ 점수 계산식 => 시간 * 0.7 + 카운트 * 0.3

2. 스코어 기능
- 타임어택 : 타이머 기능으로 최단 시간 기록 갱신

- 이동횟수 : 카운트 기능으로 클릭할 때마다 점수 깎기 (잘못된 클릭, 이동할 수 없는 영역 클릭 시 
카운트 X + “움직일 수 없습니다.”, “잘못된 행동입니다.” 등 오류 메세지 출력)
- 최고기록(스코어 보드) 표시
→ CSV 파일 연결해서 이전 최고 기록보다 높다면 갱신

3. 움직일 수 있는 영역 표시
→ 움직일 수 있는 퍼즐 색깔 바꾸기

4. 퍼즐을 움직일 때 슬라이딩 에니메이션 추가하기
→ 구현 : d(이동 거리) / r(반복 횟수) 프레임 만큼 timer를 통해 퍼즐을 이동 시켜 슬라이드 하는 것처럼 보이게 하기

5. 보드 생성 : 클리어가 불가능한 보드 생성 방지
